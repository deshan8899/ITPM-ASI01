# Transliteration Accuracy Testing using Playwright

## Module

IT3040 – ITPM (Semester 1)
BSc (Hons) in Information Technology – Year 3

---

## Assignment Overview

This project evaluates the accuracy of a Sinhala chat transliteration system that converts Singlish input into Sinhala output.

The application tested in this assignment is:
https://www.pixelssuite.com/chat-translator

The main purpose of this assignment is to identify incorrect conversions and analyze the weaknesses of the system using structured test cases and automation.

---

## Objectives

* Test the accuracy of Singlish to Sinhala transliteration
* Identify 50 failing test cases
* Automate the testing process using Playwright
* Record execution results in an Excel file
* Analyze different Singlish input types

---

## Setup Instructions

### Install Python

Download and install Python 3.11 or 3.12 from the official website:
https://www.python.org/

---

### Install Required Packages

Open Command Prompt and navigate to the project folder:

```bash
cd /d D:\test_automation
```

Run the following commands:

```bash
python -m pip install --upgrade pip
pip install playwright openpyxl
playwright install
```

---

## Running the Automation Script

Execute the following command:

```bash
python test_automation.py --excel "test_automation/Assignment 1 - Test cases.xlsx" --url "https://www.pixelssuite.com/chat-translator" --wait-ms 5000 --type-delay-ms 80 --slow-mo-ms 200 --save-every 1 --keep-open
```

---

## Excel File Structure

The Excel file contains the following columns:

* TC ID
* Input length type (S / M / L)
* Input (Singlish text)
* Expected output (Sinhala text)
* Actual output (auto-generated)
* Status (auto-generated)
* Singlish input types covered
* Evidence or rationale

Note:
Actual output and Status columns should not be filled manually. They will be updated automatically after running the script.

---

## Test Case Design

* Total number of test cases: 50
* All test cases are negative scenarios (expected to fail)
* Covers multiple Singlish input types such as:

  * Question forms
  * Commands
  * Slang expressions
  * English mixed inputs
  * Emojis
  * Numbers, dates, and time formats

---

## Output and Results

After executing the automation script, the Excel file will be updated with:

* Actual output generated by the system
* Status indicating Pass or Fail

These results are used to evaluate the accuracy of the transliteration system.
